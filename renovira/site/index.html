📝 So geben Sie Feedback

  1. Version auswählen — Klicken Sie auf die aktuelle Version (oben). Die ältere Version dient als Vergleichs-Referenz.
  2. Seite öffnen — Klicken Sie auf eine Seite oben in der Navigation. Alle 15 Seiten sind über das Menü erreichbar; die 10 Leistungen finden Sie im Dropdown unter „Leistungen".
  3. Kommentar hinterlassen — Neben jedem Textblock sehen Sie „✏️ Vorschlag". Klicken Sie darauf, schreiben Sie Ihren Kommentar und klicken Sie auf „Speichern".
  4. Mehrere Seiten kommentieren — Kommentare werden seitenübergreifend gespeichert. Sie können beliebig zwischen den Seiten wechseln.
  5. Feedback absenden — Oben rechts: „📥 Feedback: X Kommentare" → klicken → wählen Sie: „Als Datei herunterladen" (eine .txt-Datei) oder „In Zwischenablage kopieren" (zum Einfügen in E-Mail oder Messenger).
  6. An uns senden — Schicken Sie die heruntergeladene Datei oder den kopierten Text per E-Mail, WhatsApp oder wie es Ihnen am liebsten ist.

Hinweis: Ihre Kommentare werden lokal in Ihrem Browser gespeichert und bleiben erhalten, auch wenn Sie die Seite schließen.
