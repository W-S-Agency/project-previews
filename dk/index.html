So geben Sie Feedback

  1. Seite öffnen — Klicken Sie auf eine Seite oben. Alle Seiten sind über die Navigation erreichbar.
  2. Kommentar hinterlassen — Neben jedem Textblock sehen Sie „✏️ Vorschlag". Klicken Sie darauf, schreiben Sie Ihren Kommentar und klicken Sie auf „Speichern".
  3. Mehrere Seiten kommentieren — Kommentare werden seitenübergreifend gespeichert. Sie können beliebig zwischen den Seiten wechseln.
  4. Feedback exportieren — Oben rechts: „Feedback: X Kommentare" → klicken → „Als Datei herunterladen". Sie erhalten eine .txt-Datei mit allen Kommentaren.
  5. Datei an uns senden — Schicken Sie die Datei per E-Mail, WhatsApp oder wie es Ihnen am liebsten ist.

Hinweis: Ihre Kommentare werden lokal in Ihrem Browser gespeichert und bleiben erhalten, auch wenn Sie die Seite schließen.
